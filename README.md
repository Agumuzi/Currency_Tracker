# Currency Tracker

一个给 macOS 用的菜单栏汇率小工具。

做这个软件的原因很简单：平时经常要看几组固定汇率，或者在网页上看到外币价格时想顺手换算一下。如果每次都去开浏览器、搜 Google、切计算器，其实挺烦的。所以就做了这个尽量轻、尽量直接的小工具。

## 现在能做什么

- 在菜单栏里查看常用货币对
- 支持手动刷新
- 支持按设置自动刷新
- 可以自定义关注的货币对
- 可以把金额换算成自己设定的基准货币
- 支持基于选中文本做快速换算

这个软件主要是给日常使用做减法，不是专业交易软件，也不是做大而全那种金融终端。

## 适合什么场景

比较适合下面这种情况：

- 长期关注几组固定汇率
- 经常会在购物网站、聊天记录、文章里看到外币价格
- 想快速知道它大概等于多少人民币 / 卢布 / 美元
- 不想每次都专门开网页查

## 为什么第一次打开会被 macOS 拦一下

因为这个软件没有走 Apple 官方的开发者签名和 notarization 分发流程。

说人话就是：  
开发者没有订阅苹果开发者服务，所以 macOS 第一次打开时，可能会把它拦下来。这是系统正常的安全机制，不是软件本身坏了，也不是中毒。

如果被拦了，按下面操作就行：

**系统设置 → 隐私与安全性 → 仍要打开**

点完之后，再打开一次，通常后面就正常了。

## 为什么用了这两个自定义 API

这个软件目前把 **Twelve Data** 和 **Open Exchange Rates** 作为可选增强数据源。

原因很简单：  
我想要的是一个对个人使用来说足够稳定、免费额度够用、而且不用把整个软件做得很重的数据方案。

如果只绑死一个源，会有两个问题：

- 某一天额度用完了，软件体验会直接掉下去
- 不同数据源各有优缺点，只用一个不够灵活

所以最后用了一个“主源 + 备用源”的思路。

### 1. Twelve Data

Twelve Data 是我优先级更高的数据源。

主要原因是它对这个软件的使用场景更合适：  
它本身支持外汇数据，免费计划提供 **每天 800 API credits**，官方支持文档也明确写了 free/basic 计划是按这个额度来的。这个软件当前的请求方式下，一次完整刷新大约会消耗 8 credits，所以实际大致可以理解为 **每天约 100 次刷新**，对个人使用已经很充足了。

它更适合放在前面，还有一个重要原因：  
Twelve Data 的定位本来就更偏“金融市场数据接口”，除了外汇，还覆盖股票、加密货币等数据，API 设计也比较适合以后继续扩展。官方把它描述为一个统一的金融市场数据平台。

对这个项目来说，Twelve Data 的优势主要是：

- 免费额度按天给，个人日常使用更从容
- 更偏实时金融数据场景
- 未来如果继续扩展别的行情能力，延展性更好

它的官网和文档入口见官方页面。

### 2. Open Exchange Rates

Open Exchange Rates 是这个项目里的第二优先级数据源，更像一个稳定的备用方案。

它免费计划的特点很明确：  
官方提供 **每月 1,000 次请求**，并且免费计划是 **按小时更新**。官方 FAQ 里还写得很具体：小时级数据通常会在每小时开始后的 1 到 2 分钟左右刷新。

它放在第二位，不是因为它差，而是因为它更适合做“稳妥兜底”：

- 免费计划规则清楚
- 汇率覆盖广
- 小时级更新对于很多非高频场景已经够用
- 作为备用源很合适

另外，Open Exchange Rates 官方也强调它覆盖 200 多种货币，并提供实时与历史汇率数据。

它的官网和说明文档见官方页面。

### 3. 为什么不是只用一个

这个软件是菜单栏小工具，不是专业交易终端。  
对我来说，最重要的不是“极限低延迟”，而是：

- 平时打开就能用
- 额度不要太容易撞线
- 某个源短时失败时不要整体验崩掉

所以最后的思路不是“选出唯一最强 API”，而是：

- **Twelve Data 负责主力刷新**
- **Open Exchange Rates 负责备用兜底**

这样做的好处很实际：

- 日常主要吃 Twelve Data 的按天额度
- 如果 Twelve Data 因为额度、失败、限流或别的问题暂时不可用，可以切到 Open Exchange Rates
- 软件整体可用性会比只绑一个源更稳

### 4. 为什么这套组合适合这个项目

因为这个项目本身就是偏个人使用的轻工具。

它不追求做成交易系统，也不追求秒级高频推送。  
它更在意的是：点开就能看、顺手能换算、长期用起来别太折腾。

在这个前提下：

- **Twelve Data** 适合当主源，因为免费额度按天给，个人体验更宽松
- **Open Exchange Rates** 适合做备用，因为规则稳定、文档清晰、小时级更新也足够覆盖大多数普通查看场景

这个组合基本就是我在“免费、够用、稳定、好维护”之间做出来的折中。
